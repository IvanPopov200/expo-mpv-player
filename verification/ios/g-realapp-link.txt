# G-realapp — a community RN native module links alongside MPVKit (static)

Reproduces the host-app condition the bare example lacked: a non-Expo RN native
module present in the same app. Under the OLD dynamic-frameworks requirement this
is exactly what failed (RCTEventEmitter / RCTRegisterModule undefined symbols).

Added: react-native-safe-area-context 5.7.0 (Obj-C native module, autolinked,
built from source). Example built for iphonesimulator, Debug, STATIC linkage.
